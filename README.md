
# AdoçãoPet - Monorepo (Frontend + Backend)

Scaffold de projeto para sistema de adoção de pets:
- Frontend: React + Vite + TypeScript + Tailwind + PWA + Firebase Auth
- Backend: Node.js + TypeScript + Express + MySQL + Firebase Admin (storage token validation)
- Estrutura pensada para escalabilidade, acessibilidade e segurança.

Instruções rápidas:
- Backend: `cd backend && npm install && npm run dev`
- Frontend: `cd frontend && npm install && npm run dev`

Preencha os `.env` a partir dos `.env.example`.
